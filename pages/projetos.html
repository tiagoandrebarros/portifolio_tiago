Gerador de onda quadrada personalizavel

Projeto freelancer (ano: 2025)

Tipo: Microcontroladores e Mobile

Esse projeto consiste em usar um Arduino Nano para gerar uma onda quadrada que serve para acionar um dispositivo externo, as características da onda quadrada como frequências, duração de cada frequência e duty são enviadas de um smartphone para o Arduino Nano via bluetooth, dessa forma é possivel criar ondas quadradas personalizadas para fazer diferentes tipos de funcionamentos do dispositivo externo.

Programa desktop para gerar relatórios de vendas e abastecimentos

Projeto freelancer (ano: 2025)

Tipo: Programa desktop para Windows

Para emitir os relatórios o usuário informa a quantidade atual dos produtos no estoque, o aplicativo calcula quando foi vendido se baseando nos dados anteriores levando em conta os abastecimentos, para gerar dois arquivos CSV na pasta Downloads, um é o relatório das vendas e o outro é o relatório de abastecimento mostrando a data do recebimento e a quantidade.
